Features

Free

This jQuery plugin is open source, under the MIT license.
Feel free to contribute on the repository.

Customizable

Use your own CSS, many options are avalaible (including effects) in the documentation.

Controls

Control the slider by clicking or pressing your keyboard keys.

Caption

Specify a caption and a link for each image to show more information.

Multiple

Create as many instances as you want on a same page.

Tests suite

A tests suite shows that the plugin passes the basics specifications.
